## 🔧 技术特性

- **前端框架**: Alpine.js 3.13.3
- **搜索引擎支持**: 百度、Bing、Google、DuckDuckGo
- **背景图片**: 自动从Bing获取每日壁纸
- **本地存储**: 支持自定义快捷网站和设置
- **响应式设计**: 支持移动端和桌面端
- **夜间模式**: 支持明暗主题切换

## 📱 使用方法

1. **搜索**: 在搜索框中输入关键词，按Enter搜索
2. **切换引擎**: 点击搜索引擎名称切换不同的搜索服务
3. **快捷网站**: 右键点击打开快捷网站面板
4. **夜间模式**: 按F9键切换明暗主题
5. **退出**: 按Esc键退出当前状态

# 简搜

一个简洁优雅的个人搜索导航页面。

## 功能特性

### 搜索
- 多搜索引擎支持（Bing、百度、Google、DuckDuckGo）
- 搜索引擎图标显示
- 实时搜索建议（JSONP 无跨域问题）
- 搜索历史记录（最多 10 条，支持单条删除和清空）
- 键盘导航（上下键选择、Enter 搜索、Esc 退出）

### 快捷导航
- 自定义快捷网站管理（增删改）
- 拖拽排序（桌面端 + 移动端触屏）
- 批量管理（全选、批量删除、恢复默认）
- 导入/导出配置
- 自动获取网站图标
- 键盘导航（方向键选择、Enter 打开、Tab 循环）

### 主题
- 日夜模式自动切换（根据日出日落时间）
- 手动切换覆盖自动模式
- 基于 IP 地理位置计算日出日落

### 背景
- Bing 每日壁纸
- 本地缓存（秒开）
- 无痛更新（后台检查新图）
- 显示图片标题和版权信息

### PWA
- Service Worker 离线支持
- 可添加到主屏幕

### UI/UX
- Tailwind CSS 样式
- 毛玻璃效果
- 弹性动画
- Toast 操作提示
- focus-visible 无障碍支持
- 响应式设计

## 技术栈

- **框架**: Alpine.js 3
- **样式**: Tailwind CSS
- **图标**: SVG（Heroicons 风格）
- **存储**: localStorage

## 快捷键

| 按键 | 功能 |
|------|------|
| `F9` | 切换日夜模式 |
| `Esc` | 退出当前状态 |
| `↑` `↓` | 选择搜索建议 |
| `Enter` | 执行搜索 |
| `Shift + Backspace` | 清空搜索内容 |
| 右键 | 打开快捷导航 |
| `← → ↑ ↓` | 快捷导航中选择网站 |
| `Tab` | 快捷导航循环选择 |
